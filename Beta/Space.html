Space Game

Engine Used: Unity

Language Used: C++

My Space Game project was my first project using Unity and it was a 2D focused game, with arcade elements. Due to the parameters of the assignment, the games focus was less on menuing and UI, and more on gameplay and mechanics. In this project I implemented 2D Sprites, animations, sounds and pitching for variation, life systems, consumables, and enemy variation.
